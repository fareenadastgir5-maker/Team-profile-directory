# Team Profile Directory

A React + Vite project built as part of the MERN Stack + AI Engineering Bootcamp.

## Project Overview

This application displays a directory of team members using reusable React components. It demonstrates:

- JSX
- Components
- Props
- Children Props
- Prop Drilling
- Array Mapping
- Conditional Rendering
- Vite

## Features

- Display 6 team member cards
- Admin badge for admin members
- Skills displayed as badges
- Team member location
- Follow button with console log
- Responsive card layout
- Clean UI

## Technologies Used

- React
- Vite
- JavaScript (ES6)
- CSS3

##  Installation

Clone the repository:

```bash
git https://github.com/fareenadastgir5-maker/Team-profile-directory/tree/main/reactfirstclass
```

Go to the project folder:

```bash
cd team-directory
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

##  Output

The application displays:

- Team Member Name
- Role
- Age
- City & Country
- Skills
- Admin Badge
- Follow Button
